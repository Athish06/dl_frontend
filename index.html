About SentimentLens

SentimentLens is a deep learning-powered sentiment analysis tool built with a hybrid CNN-BiLSTM architecture. It combines Convolutional Neural Networks for extracting local text patterns (n-grams) with Bidirectional LSTM for understanding sequential context — achieving higher accuracy than either approach alone.

🔍 CNN Layer

Multi-kernel Conv1D (3,5,7-gram) with padding preserves the full sequence length, extracting local n-gram patterns without collapsing the timeline.

🔄 BiLSTM Layer

Bidirectional LSTM reads the full 200-step CNN feature sequence in both directions, capturing complete forward and backward context.

📊 Dataset

Trained on a mixed 250K dataset — 50,000 IMDB movie reviews + 200,000 Yelp reviews — with binary sentiment labels (Positive / Negative).

⚡ Stack

PyTorch backend, FastAPI server, vanilla HTML/CSS/JS frontend. No heavy frameworks needed.
